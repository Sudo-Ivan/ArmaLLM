# ArmaLLM and Arma Dataset

A collection of tools and datasets for Arma 3 and Arma Reforger. For fine-tuning using unsloth on Llama, Qwen, Mistral, and more.

## Data Tools

### Requirements

- Python 3.10+
- pip
- poetry
- playwright

### Installation

```bash
poetry install
```

### Usage

```bash
poetry run python data-tools/data-scraper.py
```
